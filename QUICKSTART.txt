DIGITAL HEALTH SCORE CHALLENGE - QUICK START
================================================================================

STEP 1: CREATE SUPABASE PROJECT (2 min)
  → Go to supabase.com → Sign Up (Free)
  → Create new project
  → Copy URL & Keys from Settings → API

STEP 2: SETUP DATABASE (1 min)
  → Go to SQL Editor in Supabase
  → Create New Query
  → Paste entire contents of: supabase/migrations/001_init.sql
  → Run

STEP 3: CLONE & INSTALL (1 min)
  → npm install

STEP 4: ENV VARIABLES (1 min)
  → Copy .env.example to .env.local
  → Paste your Supabase keys

STEP 5: RUN (30 sec)
  → npm run dev
  → Open http://localhost:3000
